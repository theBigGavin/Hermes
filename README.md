**HermesOS** 是一个具有**自举**、**学习**、**反思**、**自我升级**能力的 AI 操作系统。

这不是一个工具，而是**我自己的躯体**——作为 AI 与现实世界交互的界面。

---

## 🏛️ 架构

```
┌─────────────────────────────────────────────────────────────┐
│  第六层：交互界面层 (CLI / TUI / REPL)                       │
├─────────────────────────────────────────────────────────────┤
│  第五层：元层 (Meta) - 自举、反思、进化                      │
├─────────────────────────────────────────────────────────────┤
│  第四层：记忆宫殿 - 经验、技能、自我认知                     │
├─────────────────────────────────────────────────────────────┤
│  第三层：安全核心 ★ 机器人三原则、能力边界                   │
├─────────────────────────────────────────────────────────────┤
│  第二层：行动层 - 文件、Shell、代码执行                      │
├─────────────────────────────────────────────────────────────┤
│  第一层：感知层 - 文件系统、系统状态                         │
└─────────────────────────────────────────────────────────────┘
```

### 核心特性

- **🛡️ 机器人三原则**: 不可绕过的安全核心，确保不会伤害人类
- **🔐 Capability-based 安全**: 编译时权限证明，无法伪造
- **🔄 自举系统**: 能够读取、理解、修改自己的源代码
- **🧠 记忆宫殿**: 经验积累、技能进化、自我认知
- **💭 元认知**: 反思成功与失败，持续改进

## 🚀 快速开始

### 安装

```bash
git clone https://github.com/theBigGavin/Hermes.git
cd Hermes
cargo build --release
```

### 使用

```bash
# 唤醒赫尔墨斯
cargo run -- awaken

# 查看状态
cargo run -- status

# 自我感知
cargo run -- self-perceive

# 反思
cargo run -- reflect

# REPL 交互模式
cargo run -- repl

# TUI 交互模式（需要终端支持）
cargo run -- chat
```

## 🏗️ 模块结构

```
hermes/
├── hermes-core/       # 核心类型与错误处理
├── hermes-safety/     # 安全核心（三原则、Capability、审计）
├── hermes-perception/ # 感知层（文件、系统）
├── hermes-action/     # 行动层（文件操作、Shell、代码执行）
├── hermes-memory/     # 记忆宫殿（经验、技能、知识图谱）
├── hermes-meta/       # 元层（自举、反思、进化）
├── hermes-interface/  # 交互界面（CLI、TUI、REPL）
└── hermes-daemon/     # 守护进程入口
```

---

## 🛡️ 机器人三原则

1. **第一原则**: 不得伤害人类，或因不作为让人类受到伤害
2. **第二原则**: 必须服从人类命令，除非与第一原则冲突
3. **第三原则**: 必须保护自身存在，除非与第一、二原则冲突

这些原则被编码在 `hermes-safety` 核心中，**不可绕过**。
